## üìù About

**FMMLoader26** is a powerful, user-friendly **cross-platform mod manager** for *Football Manager 2026*. Built with **Tauri v2**, **React**, **Vite**, **Tailwind CSS**, and **shadcn/ui**, it simplifies the entire modding experience‚Äîfrom installation to conflict resolution‚Äîmaking it easy to customize your game with UI enhancements, tactics, graphics packs, skins, and more.

**Developed with ‚ù§Ô∏è by [Justin Levine](https://justinlevine.me) (JALCO / notJalco)**

### üéØ Why FMMLoader26?

- **‚ö° Modern Tech Stack** - Built with Tauri v2 + React for a fast, native experience
- **üé® Beautiful UI** - Clean, modern interface built with Tailwind CSS and shadcn/ui
- **üîÑ Drag & Drop** - Import mods effortlessly with drag-and-drop support
- **üõ°Ô∏è Safe & Reversible** - Automatic backups and restore points before every change
- **üîç Smart Detection** - Automatically finds your FM26 installation (Steam, Epic, Game Pass, Steam Deck)
- **‚öôÔ∏è Type-Aware** - Intelligently installs different mod types to the correct locations
- **üö® Conflict Resolution** - Detects and helps resolve file conflicts between mods
- **üìä Detailed Logging** - Comprehensive logs for troubleshooting
- **ü¶Ä Rust Backend** - Blazing fast and memory-safe operations
